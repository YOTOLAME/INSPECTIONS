Inspection Details

15.1 -- General
15.1.1 Does the vessel have on board a copy of the most recent DP FMEA (DP FMEA Desktop analysis)?
Record the date of the report and the name of the issuing company.
15.1.2 Is the latest DP FMEA revision less than 5 years old and/or has there been a third party gap analysis been completed within the last 5 years stating the DP FMEA is still valid?
Both IMO and IMCA guidance state that formal review of the DP FMEA shall be completed within a 5-year period.
IMO MSC.1/Circ.1580; IMCA M 166; IMCA Information Note 1605
15.1.3 Have any modifications been sighted/reported to the DP mission critical equipment since the last DP FMEA revision. If applicable, have any modifications been proven by testing?
IMCA M 166 states: "The FMEA will become out of date if it is not maintained regularly and systematically, due to changes in operating procedures, modifications to DP hardware and software, to confirm compliance with the latest industry guidance, etc., over the life cycle of the vessel. A systematic FMEA review through the vessel's life cycle should be an ongoing process which should be formally completed at least once every five years."
IMCA M 166, Chapter 2.7 Updating of an FMEA
15.1.4 Is a copy of the latest DP FMEA Proving Trials (5-yearly DP FMEA verification trials) available on board and is it less than 5 years old?
15.1.5 Are the latest revisions of the FMEA Report and DP FMEA Proving Trials approved by class?
15.1.6 Have the findings (if any) from the latest DP FMEA Proving Trials been addressed?
Record the date and issuing company of the DP FMEA proving trials.
15.1.7 Does the vessel have on board a copy of the most recent DP Annual Trials report?
Record the date of the report, issuing company and specify if the DP annual trials were performed remotely, semi-remotely or as an on-board trial. The DP annual trials should be conducted as per IMCA M190.
15.1.8 Have the latest DP Trials been completed within the window?
If there has been no DP Annual trials and or proving trials within the last 15 months, leave an observation. If there has not been DP trials in the last 12 months please comment. The Annual DP trials and or DP Proving trials should be carried out within three (3) months before or after each anniversary date of the Dynamic Positioning Verification Acceptance Document.
IMCA M190; IMCA M103
15.1.9 Have recommendations from the annual DP trial report been addressed and closed out as required?
If there are any outstanding A/B findings, record it as an observation and provide detail.
15.1.10 Have all personnel involved in DP operations read and understood the DP FMEA?
Confirm that the DPO's have signed an acknowledgement form.
15.1.11 Is there a valid software maintenance agreement(s) in place?
The following systems to be covered: DP control system; Power system; Thruster system. The software maintenance agreement should cover safety and security patches, obsolescence management, incident reporting and response handling, software change handling and software asset inventory. A statement of product status and maintainability may be accepted as an alternative.
DNV-RP-0582 and IMCA M 103
15.1.12 Do the vessel procedures require a minimum of two DP operators to be on duty during DP operations?
Check that there is sufficient qualified DPOs to provide rest/meal relief (if applicable).
15.1.13 Is the bridge access restricted to only essential personnel during DP operations?
Check that bridge restrictions are implemented - i.e. by sign, procedures or standing orders.
15.2 -- Operations
15.2.1 In the last 12 months has the vessel operated without experiencing any loss of position incidents?
If the vessel has recorded any Yellow or Red condition DP alarms. Inspector to note and record all such incidents.
15.2.2 Is the vessel without any defects which result in a reduction of DP Capability and/or redundancy?
Record any DP mission critical equipment which is not operating as expected.
15.2.3 Does the vessel use the IMCA Incident reporting system?
Both Members and non-Members of IMCA are asked to submit reports on the following: DP Incident; DP Undesired Event; DP Observation. If recent reports are not available check that the vessel's management system requires IMCA DP incident reporting.
15.2.4 Does the vessel carry out risk assessments for specific operations (as listed in guidance)?
Risk assessments shall cover: SIMOPS; External loads such as tuggers; Weather related contingencies, etc.; Weather side working; Environmental / Shallow water operation; Heavy lift operation.
15.2.5 Are Manual thruster control levers and emergency stops located within easy reach of the DPO and are each emergency stop well protected against inadvertent operation?
For vessels build after June 2017, IMO MSC.1 Circ. 1580, Section 3.3.6 states that "individual thruster emergency stop systems should be arranged in the DP control station". Regardless of above guidance, inspector should write an observation if each thruster does not have an individual emergency stop and/or emergency stops / control levers are not within easy reach of DPO.
IMO MSC.1 Circ. 1580
15.2.6 Can the health of the position reference systems be monitored by the DPO, independently of the DP control station?
Inspector to comment if DPO needs to leave the immediate vicinity of the DP desk to view health information.
15.2.7 Does the vessel have a vessel specific DP Operation manual on board and is there evidence that relevant DP personnel has read the document?
IMCA M109; IMO Resolution 645 (Operational Requirement); IMO MSC.1/Circ. 1580
15.2.8 Does the DP operations procedure include specific guidance for any DP specific operations that the vessel will conduct, e.g. Anchor Handling, Towing, Heavy Lift, Diving, Pipelay, etc?
Procedure should address when and where DP is used and the checks required prior to use.
OCIMF Dynamic Positioning Assurance Framework Information paper
15.2.9 Is there a competence scheme for key DP personnel onboard?
This could be a in-house training scheme.
15.2.10 Are DP checklists in place to cover bridge, engine room and electrical systems for DP operations?
Checklists can include DP setup (Bridge and engine) / 500 m safety zone entry (Bridge and engine) / Field arrival / DPO and engine room periodical handovers.
15.2.11 Are DP Capability Plots in place to cover the normal and expected operations, and worst case failure?
IMCA M 140 "Specification for DP Capability Plots"; Marine Technology Society DP Operations Guidance
15.2.12 Are DP footprints regularly recorded and compared against previous footprints and the DP Capability Plots?
Check that the operator has guidance for recording DP footprints and specifies frequency of recording.
IMCA M 140 - Specification for DP Capability Plots and MTS Operations Guidance DP Footprint Plots
15.2.13 Depending on vessel activity and if required, are Activity Specific Operating Guidelines (ASOG) or Well Specific Operations Guidelines (WSOG) or Field Specific Operations Guidelines (FSOG) in place and utilised?
Check that: All relevant parties are aware of the specific operating guidelines (ASOG, CAMO, TAM etc); There are defined limits set for Green, yellow and red DP operations; DP cessation triggers are clearly defined within specific operating guidelines; Consequence analysis alarms are used as an input to the specific operating guidelines. Inspector to record an observation if specific operating guidelines are not readably available at DP control station.
15.2.14 Is the DP control console located so that the DPO can also observe the controls, the external environment and the working operations of the vessel/unit?
If 'No', state whether CCTV is utilised.
IMCA M 103
15.2.15 Is the DP alert triggering system in immediate reach of the DPO at console?
The DP Alert System is used during critical operations such as diving, drilling, pipelaying, walk-to-work or other critical operations and includes traffic lights.
15.3 -- Equipment
15.3.1 Is the Dynamic Positioning control systems in good order?
Note any recommendations/deficiencies reported.
15.3.2 Are all position reference systems in good order and are both relative and absolute position references available on the vessel for the applicable operations vessel likely to conduct?
IMCA M 252 summarises best practice for the selection and use of position reference systems (PRS) and sensors in dynamic positioning (DP) control systems.
IMCA M 252
15.3.3 If a DP data recorder is not fitted, confirm that procedures are in place for securing relevant data in the event of a DP incident?
IMCA M 103
15.3.4 Is there a procedure and evidence of the regular checking of the secure power supply systems (UPS Battery systems)?
Planned maintenance routines should exist for the UPS/s.
IMCA M 190
15.3.5 Does the DP system have a continuous analysis function checking that in terms of thruster and power can maintain position after the worst case failure (consequence analysis function)?
IMCA M 103
15.3.6 Is the DP control system fitted with real time DP capability envelopes or does the vessel have procedure to do a drift off test prior to DP operation?
Inspector to record in comments if active monitoring or drift off test is used.
15.3.7 Is the actual bus bar configuration in accordance with the FMEA and the last DP trials performed?
Inspector should state whether the bus ties are open or closed configuration for each mode of DP operation (i.e. Mode 1, 2 etc).
15.3.8 Is the DP control system included within the Planned Maintenance System?
Including all position reference systems, UPSs and sensors.
15.3.9 If unit is a DP flotel fitted with a connected gangway, confirm that the gangway is used as Positioning Reference System (PRS)?
Record an observation if gangway is not used as a PRS.
15.3.10 If the unit is a DP flotel operating on a turret moored FPSO or any moving asset with connected gangway, confirm that DP system is fitted with "follow target" mode option?
Record an observation if "follow target mode" is not available.
15.4 -- Competence
15.4.1 Are the vessels crew suitably qualified for DP Operations?
All key DP personnel (including Engineer and/or Electro-technical personnel) onboard should comply with the IMCA minimum requirements for experience and training. Confirm DP certification been issued by a recognised body, such as the Nautical Institute. Experience should be recorded in DP log books.
IMCA M 117
15.4.2 Is there an Engineer and or Electronic Technician on-board with approved training on the DP system?
Confirm that Engineers and ETOs have appropriate training/guidance on how to operate/maintain plant when in DP mode. Sight certificates / records of DP system training from system manufacturer.
IMCA M117 section 6
15.4.3 Are DP drills included in the vessel drill matrix and performed regularly?
Inspector to check drill record and record date of last DP drill.
